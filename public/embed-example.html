WebFeedback Widget Embed Example

This page demonstrates how to embed the WebFeedback widget using a script tag.

How to Embed

Add this script tag to your HTML, replacing YOUR_API_KEY with your actual API key:

<script src="https://yourdomain.com/widget/v1/loader.js?key=YOUR_API_KEY"></script>

Example with API Key

<script src="https://yourdomain.com/widget/v1/loader.js?key=wf_abc123xyz"></script>
Note: Replace https://yourdomain.com with your actual domain where the widget is hosted.

Getting Your API Key

To get an API key:

  1. Go to the admin dashboard
  2. Create a new customer/project
  3. Enter your GitHub credentials (token, owner, repo)
  4. Add allowed domains (e.g., example.com, www.example.com)
  5. Copy the generated API key and script tag

Domain Whitelisting

For security, you must whitelist the domains where the widget will be used. Only requests from whitelisted domains will be accepted.

When creating a customer, specify allowed domains like:

  • example.com
  • www.example.com
  • app.example.com

Troubleshooting

  • Widget not loading: Check that your API key is correct and the domain is whitelisted
  • 403 Forbidden: Your domain is not in the allowed domains list
  • 401 Unauthorized: Invalid or inactive API key
  • Console errors: Check browser console for detailed error messages
